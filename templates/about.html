About

Summary

This application is another one of my side projects, it is essentially a task management productivity tool. It was built to experiment with interactions between a NOSQL database and a dynamic web application. Previously, I've built web applications using a relational database backend and wanted to get a feel architecting and building an application depending on a NOSQL backend. This application simplifies the process of working on projects/tasks so that the user has more time to get things done, while minimizing

NOSQL has some notable advantages over SQL, in that it:

  • Scales well horizontally.
  • Relies on a document style system, similar to hashtable structure. This means that little to no transforming needs to be done in between reading and rendering data.
  • Need for complex joins are eliminated.
  • Faster development time.

Routes

Listed below are the routes included in this application. All routes except for /login and /register are protected and cannot be accessed unless user is authenticated with valid credentials.

  • /home or / - Allows registered members to create, view and manage tasks.
  • /login - Members can login to the application here.
  • /register - New members can sign up an account here.
  • /members - Lists all registered members.
  • /features - Lists features available and to be built out in the future.
