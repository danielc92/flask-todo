Features Page

This page lists features that have already been built into the application, as well as features I wish to add in the future.

Current Features

This applications intent is to drive productivity through a simple and easy to GUI and intuitive design. The following features are currently included;

  • Registration and Login system
  • Password SHA-512 encryption
  • NOSQL database system, using MongoDB
  • Dynamic routing
  • Pre-request authentication
  • Secure session handling
  • Creation of tasks, including descriptions and automatic timestamps
  • Members page listing all members signed up
  • Board page including all tasks within user scope
  • Features page including current and future features.
  • Task status can be updated from incomplete to complete via button

Future Considerations

Features I would like to build out in the future;

  • Deleting tasks within user scope
  • Breaking down tasks into micro-tasks
  • More detailed options for tasks (value and time metrics)
  • Profile page
